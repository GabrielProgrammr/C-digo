Estrutura de Dados


Estrutura de Dados é uma estrutura organizada e de dados na memória de um computador ou em qualquer dispositivo de armazenamento, de forma que dados possam ser utilizados de forma correta.

Operações Básicas


Em uma estrutura de dados devemos saber como realizar um determinado conjunto de operações básicas, como por exemplo:

  • Inserir dados
  • Excluir dados
  • Localizar um elemento
  • Percorrer todos os itens constituitens da estrutura para visualização
  • Classificar, que se resume em colocar os itens de dados em uma determianda ordem

Principais Estruturas de Dados

  • Veteores e Matrizes
  • Registro
  • Lista
  • Pilha
  • Fila
  • Árvore
  • Tabelha Hash
  • Grafos

    • Algoritmo


    Um algoritmo é um conjunto de instruções estruturadas e ordenadas, seu objetivo é realizar uma tarefa ou operação específica. Os algoritmos são utilizadas para manipular dados nas estruturas de várias formas, como por exemplo:

    • Inserir
    • Excluir
    • Procurar e ordernar dados, somar, calcular números.
